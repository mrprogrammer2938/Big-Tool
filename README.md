Big-Tool tools:
- nmap

- cewl

- wifite

Big-Tool package:

The install.sh file runs with root !

### run root...
```
sudo su
```

**installing**
```
git clone https://github.com/msprogrammer2938/Big-Tool.git

cd Big-Tool

bash install.sh

./tools
```
## update Big-Tool
```
cd Update

./update
```
